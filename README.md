# vibe-components

A small Vite + React + TypeScript component gallery that uses React Router's declarative mode.

## Development

```bash
npm install
npm run dev
```

## Available components

- `src/components/button`
- `src/components/toggle`

Each component lives in its own folder with component code, styles, and an index export so it can be copied into another project with minimal changes.

## Deployment

GitHub Pages deployment is configured in `/home/runner/work/vibe-components/vibe-components/.github/workflows/deploy.yml`.
