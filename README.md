# vibe-components

A small Vite + React + TypeScript gallery for copy-friendly components and pages that uses React Router's declarative mode.

## Development

```bash
npm install
npm run dev
```

## Available components

- `src/components/button`
- `src/components/checkbox`
- `src/components/toggle`

Each component lives in its own folder with component code, styles, and an index export so it can be copied into another project with minimal changes.

## Available pages

- `src/pages/todo-app`

Pages are small composed examples built from the reusable components in this repository.
